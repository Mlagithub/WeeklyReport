  <action>
Create pyproject.toml at project root with the following exact configuration:

```toml
# Project configuration for linting and formatting
# Replaces separate .flake8, setup.cfg files per modern Python standards

[tool.ruff]
line-length = 120
target-version = "py312"

[tool.ruff.lint]
select = [
    "E",   # pycodestyle errors
    "F",   # pyflakes
    "W",   # pycodestyle warnings
    "I",   # isort
    "UP",  # pyupgrade
    "B",   # flake8-bugbear
]
ignore = ["E501"]  # Line length handled by formatter

[tool.ruff.lint.isort]
known-first-party = ["app", "models", "routes", "forms", "utils", "extensions", "config", "exporters"]

[tool.ruff.lint.per-file-ignores]
"tests/*" = ["F401", "F811"]  # Allow unused imports and redefinitions in tests

[tool.black]
line-length = 120
target-version = ["py312"]
```

This configuration:
- Uses ruff as unified linter (replaces flake8, isort, pydocstyle)
- Sets line-length=120 consistent with existing code style
- Targets Python 3.12 (current environment)
- Enables import sorting with known-first-party modules
- Allows test-specific conventions (unused imports for fixtures)
