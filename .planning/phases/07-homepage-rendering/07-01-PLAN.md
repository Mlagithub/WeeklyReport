ALLOWED_TAGS = {
    'p', 'br', 'b', 'i', 'strong', 'em', 'u',
    'ul', 'ol', 'li', 'a', 'img',

    'a': ['href', 'title', 'target', 'rel'],

3. Add the template filter registration AFTER the `create_app` function but BEFORE `app = create_app()` (around line 67):
```python
# Register custom Jinja2 filters
@app.template_filter('sanitize_html')
def sanitize_html(text):
    """Sanitize HTML content for safe rendering.

    Allows common CKEditor formatting tags while blocking XSS.
    Used for RENDER-01 (rich text) and RENDER-02 (XSS prevention).

    Args:
        text: HTML string to sanitize.

    Returns:
        Sanitized HTML string, safe for |safe filter.
    """
    if not text:
        return ''
    return bleach.clean(
        text,
        tags=ALLOWED_TAGS,
        attributes=ALLOWED_ATTRIBUTES,
        protocols=ALLOWED_PROTOCOLS,
        strip=False
    )
```

Note: The filter must be registered AFTER `app = Flask(__name__)` is created, but the decorator `@app.template_filter()` works because `app` is defined later in the file. Place this function immediately before `app = create_app()` so it's defined when the app instance is created.</action>
