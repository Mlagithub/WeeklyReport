## OpenAI Chat Completions API Format

POST to {api_url}/chat/completions:
```json
{
  "model": "{model_name}",
  "messages": [{"role": "user", "content": "{prompt}"}]
}
```
Response format:
```json
{
  "choices": [{"message": {"content": "AI response"}}]
}
```
