# Phase 18 Plan 01: Text Polish Feature

## Objective
Add AI-powered text polish feature to report editor with admin-configurable default prompt.

## Context
- **Reuses:** `call_ai_api` from ai_utils.py, AIConfig model from Phase 14
- **Integration:** create_records.html editor, edit_record route, config.html settings

## Tasks

### Task 1: Add polish_prompt to AIConfig model
**Type:** auto
**File:** models.py
**Action:** Add `polish_prompt` column to AIConfig model (nullable, defaults to standard polish prompt)
**Verification:** Model has polish_prompt field, migration handled

### Task 2: Add PolishForm and update AIConfigForm
**Type:** auto
**File:** forms.py
**Action:**
- Add `polish_prompt` TextAreaField to AIConfigForm
- Add PolishForm with `text` TextAreaField for polish requests
**Verification:** Forms defined with correct fields

### Task 3: Add /polish-text route
**Type:** auto
**File:** routes.py
**Action:**
- Add POST /polish-text route
- Get text from request, get polish_prompt from AIConfig (or use default)
- Call `call_ai_api(user_id, "polish", prompt)`
- Return JSON with polished text
**Verification:** Route returns polished HTML content

### Task 4: Add Polish button to editor templates
**Type:** auto
**Files:** templates/create_records.html
**Action:**
- Add "AI润色" button next to editor
- Add JavaScript to call /polish-text with editor content
- Replace editor content with polished result
**Verification:** Button appears, clicking sends content, result replaces editor

### Task 5: Update config.html for polish prompt
**Type:** auto
**Files:** templates/config.html
**Action:**
- Add polish_prompt field to AI config form section
- Show default prompt as placeholder
**Verification:** Admin can set polish prompt

## Output Spec
- AIConfig.polish_prompt field exists
- /polish-text route works
- Polish button in editor calls AI and replaces text
- Admin can configure default polish prompt

## Success Criteria
1. User can click "Polish" button in report editor
2. Polished text replaces editor content
3. Admin can configure default polish prompt in settings
