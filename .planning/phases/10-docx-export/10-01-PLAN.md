    1. `_extract_images(self, html: str) -> List[Tuple[str, str]]`:
       - Parse HTML with BeautifulSoup
       - Find all img tags with src attribute
       - Return list of (element_id, src_url) tuples

    2. `_embed_images(self, document: Document, images: List[Tuple[str, str]]) -> None`:
       - For each (placeholder_id, url):
         - Use ImageResolver.get_image_bytes(url) to get image data
         - If image exists: find placeholder in document, add picture
         - If image missing: replace with "[图片缺失]" text
       - Use python-docx's document.add_picture() with BytesIO

    Implementation approach:
    ```python
    from bs4 import BeautifulSoup
    from io import BytesIO

    def _extract_images(self, html: str) -> Tuple[str, List[Tuple[str, str]]]:
        """Extract images from HTML and replace with placeholders.

        Returns:
            Modified HTML with placeholders, list of (placeholder_id, src_url)
        """
        soup = BeautifulSoup(html, 'html.parser')
        images = []
        for i, img in enumerate(soup.find_all('img')):
            src = img.get('src', '')
            if src:
                placeholder = f'[[IMAGE_PLACEHOLDER_{i}]]'
                img.replace_with(soup.new_string(placeholder))
                images.append((placeholder, src))
        return str(soup), images

    def _embed_images(self, document: Document, images: List[Tuple[str, str]]) -> None:
        """Embed images into document at placeholder positions."""
        for placeholder, url in images:
            image_bytes = self.image_resolver.get_image_bytes(url)
            if image_bytes:
                # Find placeholder in document and replace with image
                for para in document.paragraphs:
                    if placeholder in para.text:
                        # Clear paragraph and add image
                        para.clear()
                        para.add_run().add_picture(BytesIO(image_bytes))
                        break
    ```

    1. Call _extract_images() before HTML conversion
    2. Convert modified HTML to DOCX
    3. Call _embed_images() after conversion
