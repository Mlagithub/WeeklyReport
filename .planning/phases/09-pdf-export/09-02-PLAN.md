2. If format == 'pdf':
   - Get all records from query (not grouped by week)
   - Use ExporterFactory.get_exporter('pdf')
   - Call exporter.export(records, title='周报', include_date=True)
   - Return send_file with PDF output

3. Keep existing Excel logic as fallback

Key implementation:
```python
from exporters import ExporterFactory

format = request.form.get('format', 'xlsx')
records = query.all()

if format == 'pdf':
    exporter = ExporterFactory.get_exporter('pdf')
    output = exporter.export(records, title='周报')

    if start_date and end_date:
        filename = f"周报_{start_date.strftime('%Y%m%d')}-{end_date.strftime('%Y%m%d')}.pdf"
    else:
        filename = f"周报_{datetime.now().strftime('%Y%m%d')}.pdf"

    from flask import send_file
    return send_file(
        output,
        mimetype='application/pdf',
        as_attachment=True,
        download_name=filename
    )

# Existing Excel logic for format == 'xlsx' ...
```
