|------|-------|-------|

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | 保存配置 (Save Configuration) |
| Secondary CTA | 测试连接 (Test Connection) |
| Card title | AI 配置 |
| Card description | 配置AI服务参数，支持OpenAI兼容API。 |
| Empty state | 显示空白表单字段，提示管理员配置API参数 |
| Field labels | API URL / API Key / 模型名称 |
| Field placeholders | https://api.openai.com/v1 / (password field, no placeholder) / gpt-4o-mini |
| Field descriptions | API Key字段描述：保存后仅显示最后4位字符 |
| Test success | 连接成功 |
| Test failure | 连接失败：[具体错误信息] |
| Validation errors | API URL不能为空 / API URL格式无效，必须以http://或https://开头 / API Key不能为空 / 模型名称不能为空 |
| Permission denied | 需要管理员权限 (flash message, redirect to home) |
| Destructive confirmation | 不适用 — 本阶段无破坏性操作 |

**Masked API Key display format:** `****abcd` (asterisks for all but last 4 characters)

## Component Inventory

### Card Structure (extends config.html)

```html
<div class="card shadow-sm mb-4">
    <div class="card-header bg-light">
        <h5 class="mb-0">{{ render_icon('robot') }} AI 配置</h5>
    </div>
    <div class="card-body">
        <p class="text-muted mb-4">配置AI服务参数，支持OpenAI兼容API。</p>
        {% if ai_config %}
        <p class="text-muted mb-3">当前API Key: <code>{{ masked_key }}</code></p>
        {% endif %}
        {{ render_form(ai_form) }}
    </div>
</div>
```

|-------|------|-------|------------|

| model_name | StringField | form-control | Required |
| test_submit | SubmitField | btn btn-outline-secondary mb-3 | Test Connection |
| submit | SubmitField | btn btn-primary | Save Configuration |

## Interaction States

### Test Connection Button

| State | Visual | Behavior |
|-------|--------|----------|
| Default | btn-outline-secondary | Enabled, clickable |
| Loading | Spinner + "测试中..." | Disabled, 30s timeout |
| Success | Flash message (success) | "连接成功" |
| Failure | Flash message (danger) | "连接失败：{error}" |

### Save Configuration Button

| State | Visual | Behavior |
|-------|--------|----------|
| Default | btn-primary | Enabled, clickable |
| Loading | Spinner + "保存中..." | Disabled, existing submit handler |
| Success | Flash message (success) | "配置已保存" + redirect |
| Failure | Flash message (danger) | "保存失败" |

### Form Validation

| Error Type | Display | Style |
|------------|---------|-------|
| Field required | Below field | .invalid-feedback |
| URL format invalid | Below field | .invalid-feedback |
| CSRF failure | Flash message | danger |

**Admin visibility:** Card visible only if `current_user.is_admin` is true. Non-admin users redirected to home with flash message.

**Responsive behavior:** Follow existing col-lg-7 centered layout pattern from config.html.
